Principes de conception

Les choix structurants de cette itération v3.

Compacité maximale

3 cartes métier par ligne, 12 visibles sur une demi-page. Hauteur réduite, padding millimétré.

Tension mise en avant

Toutes les cartes en tension apparaissent en brique, avec badge dédié. Tri par défaut sur la tension.

Sidebar dépliable

Six accordéons sur les fiches détaillées. L'utilisateur ouvre uniquement ce qui l'intéresse.

Typo éditoriale

Spectral italique sur les titres (esprit guide d'orientation papier), Inter pour le corps.

Pastels Wes Anderson

Crème, brique, canard, forêt, moutarde. Aucune couleur saturée, contraste doux.

Performance native

Aucun JavaScript, pas de framework. Polices et icônes en CDN. Ouverture instantanée.

Comment partager cette maquette

Cette page est l'entrée recommandée pour découvrir le portail. Tous les liens internes sont fonctionnels — votre destinataire peut naviguer comme dans le vrai site, depuis n'importe quel navigateur, sur n'importe quel appareil.
